# Currency Converter

A **React** application to convert currencies in real-time using the [ExchangeRate API](https://www.exchangerate-api.com/).  
Includes animated loading using [animate.css](https://animate.style/) and styled with [Ant Design](https://ant.design/).

## Features
- Convert between multiple currencies (USD, AZN, EUR, etc.)
- Real-time exchange rates from ExchangeRate API
- Animated loading with fadeIn/fadeOut
- Swap currencies functionality
- Responsive design
